## Como clonar e importar

- Faça um fork do projeto
- Abra o terminal do Visual Studio Code
- Digite (troque cerqueirav pelo nome do seu usuário): git clone https://github.com/cerqueirav/PooLab
- Nome da pasta: PooLab
